# 🎓 Student Result Management System

A **Python + Pandas based console application** for managing student academic records, calculating results, and generating performance reports efficiently.

---

## 🚀 Features

* ➕ Add, update, and delete student records

* 📊 Automatic percentage calculation

* 🏆 Overall & class-wise topper analysis

* 📈 School rank list & class-wise ranking system

* 📚 Subject-wise topper identification

* 📄 Student report card generation

* ⚖️ Pass / Fail / Compartment evaluation

* 💾 Data storage using Excel & CSV support

---

## 🛠️ Tech Stack

* Python 🐍

* Pandas 📊

* Excel (openpyxl engine)

* OS module

## ▶️ How to Run

1. Clone the repository:

```bash

git clone https://github.com/Bhola001/student-record-management.git
```

2. Install required library:

```bash

pip install pandas openpyxl

```

3. Run the program:

```bash

python main.py

```
